# bit-react-tutorial

このリポジトリには、Bitを使ったReactアプリのハンズオンです。

Atomic Designの設計に則り、簡単な画面の作成手順を紹介します。

## 概要

このハンズオンでは、Atomic Designの設計に則り、コンポーネントを作成していきます。
